How to Contribute

  1. 1

    Fork the Repo

    Start by forking this project to your own GitHub account.

  2. 2

    Create a Branch

    Create a new branch for your changes: `git checkout -b feature/your-feature`.

  3. 3

    Make Changes

    Implement your feature or fix, and then commit your changes.

  4. 4

    Submit a PR

    Push your branch and open a pull request to the main repository.
